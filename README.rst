Contribute
----------

* Install dependencies to hack this project.

.. code:: bash

    make install-dev


* Setup database used to run test

.. code:: bash

    make setup


* Setup pre-commit hooks (Optionnal)

.. code:: bash

    pre-commit install --install-hooks


You can use `-n` option to `git commit` command to not call hooks
